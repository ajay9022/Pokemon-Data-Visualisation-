# Pokemon-Data-Visualisation

The Visualisation has been performed on the dataset : https://www.kaggle.com/abcsds/pokemon at kaggle.

Various Types of plots using Matplotlib and Seaborn.

Plots drawn are :
1. Bar Plot
2. Box Plot 
3. Scatter Plot
4. Line Plot
5. Heat Plot
6. Dist Plot
7. Joint Plot
8. Pair Plot 
9. LM Plot
